Exercise 1 (Convex Examples).

Prove the following functions are convex

  1. (i)

    affine linear functions, i.e. f⁢(x)=aT⁢x+c for a∈ℝd, c∈ℝ,

    Solution.

    We have

    f⁢(λ⁢x+(1-λ)⁢y) =aT⁢(λ⁢x+(1-λ)⁢y)+(λ+1-λ)⏞1⁢c
    = λ⏟(a^Tx+c)_f(x) +(1-λ)⏟(a^Ty+c)_f(y). ∎
  2. (ii)

    norms, i.e. x↦∥x∥,

    Solution.

    We have

    ∥λ⁢x+(1-λ)⁢y∥⁢≤Δ⁢∥λ⁢x∥+∥(1-λ)⁢y∥⁢=scaling⁢λ⁢∥x∥+(1-λ)⁢∥y∥.∎
  3. (iii)

    sums of convex functions fk, i.e. f⁢(x)=∑k=1nfk⁢(x),

    Solution.

    We have

    f⁢(λ⁢x+(1-λ⁢y))=∑k=1nfk⁢(λ⁢x+(1-λ)⁢y)⏟≤λ⁢fk⁢(x)+(1-λ)⁢fk⁢(y)≤λ⁢∑k=1nfk⁢(x)⏟=f⁢(x)+(1-λ)⁢∑k=1nfk⁢(y)⏟=f⁢(y).∎
  4. (iv)

    F⁢(x):=supf∈ℱ⁡f⁢(x) for a set of convex functions ℱ.

    Solution.

    We have

    F⁢(λ⁢x+(1-λ)⁢y) ≤supf∈ℱ⁡λ⁢f⁢(x)+(1-λ)⁢f⁢(y)≤supf∈ℱ⁡λ⁢f⁢(x)+supg∈ℱ⁡(1-λ)⁢g⁢(y)
    =λ⁢F⁢(x)+(1-λ)⁢F⁢(y).∎
